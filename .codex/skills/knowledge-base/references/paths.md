# knowledge-base パス

## 主な場所

- `knowledge/inbox/`: 生メモと日誌
- `knowledge/inbox/done/`: distill 済みの元メモ
- `knowledge/notes/`: PREPノートとアトミックノート
- `knowledge/maps/`: テーマ別 MOC
- `knowledge/maps/monthly/`: 月次 MOC
- `knowledge/maps/mermaid/`: 図解やフロー中心の MOC
- `knowledge/reference/`: 人間向け運用説明
- `knowledge/resources/`: 参照用ストック
- `knowledge/templates/`: 学習テンプレート（`template-prep-prompt.md` はAI生成指示、`template-note.md` は手動作成・完成形確認用）
- `knowledge/projects/`: ローカル管理のプロジェクトノート

## 検索優先順
