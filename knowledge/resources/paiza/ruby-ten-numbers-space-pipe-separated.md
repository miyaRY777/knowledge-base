# 10個の数値を半角スペース+パイプ区切りで出力

難易度: D
問題ID: stdout_primer__specific_split_step5

問題文:
10 個の数値が改行区切りで与えられます。これらの数値を半角スペース 2 つとバーティカルライン | 区切りで出力してください。ただし、末尾には改行を出力してください。

入力される値:
N1

N2

N3

...

N10

期待する出力:
答えの数値を出力してください。

N1 | N2 | N3 | ... | N10

末尾に改行を入れ、余計な文字、空行を含んではいけません。

答え:

```ruby
numbers = 10.times.map { gets.chomp }

puts numbers.join(" | ")
```

解説:
`10.times.map { gets.chomp }` で、改行区切りで入力される10個の数値を1行ずつ受け取り、配列 `numbers` に入れています。
`join(" | ")` は、配列の中身を `半角スペース + | + 半角スペース` でつなげるための処理です。
`puts` を使うことで、最後に改行も出力されます。
