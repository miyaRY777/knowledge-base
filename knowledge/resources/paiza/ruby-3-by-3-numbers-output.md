# 9個の数値を3行3列で出力

難易度: D
問題ID: stdout_primer__2dim_array_step2

問題文:
9 個の数値が半角スペース区切りで入力されます。この数値を 3 行 3 列の形式で出力してください。

具体的には、入力された数値を N_1, N_2, N_3, ..., N_9 とするとき、1 行目には N_1 から N_3 を、2 行目には N_4 から N_6 を、3 行目には N_7 から N_9 を出力してください。

ただし、数値の間には半角スペースを、各行の末尾には、半角スペースの代わりに改行を入れてください。

入力される値:

```text
N_1 N_2 N_3 ... N_9
```

期待する出力:

```text
N_1 N_2 N_3
N_4 N_5 N_6
N_7 N_8 N_9
```

答え:

```ruby
numbers = gets.split

puts numbers[0..2].join(" ")
puts numbers[3..5].join(" ")
puts numbers[6..8].join(" ")
```

解説:
`gets` で入力を1行受け取り、`split` で半角スペースごとに区切って配列にします。

たとえば、入力が `0 1 2 3 4 5 6 7 8` の場合、`numbers` は次のようになります。

```ruby
["0", "1", "2", "3", "4", "5", "6", "7", "8"]
```

`numbers[0..2]` は最初の3つ、`numbers[3..5]` は次の3つ、`numbers[6..8]` は最後の3つを取り出します。

`join(" ")` を使うことで、配列の中身を半角スペース区切りの文字列に戻して出力できます。
