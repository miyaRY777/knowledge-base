# N×N の九九表を出力

難易度: C
問題ID: stdout_primer__2dim_array_boss

問題文:
自然数 N が入力されます。N 行 N 列の九九表を出力してください。具体的には、出力の i 行 j 列目 (1 ≦ i, j ≦ N ) の数値は i * j になるように出力してください。
ただし、数値の間には半角スペースを、各行の末尾には、半角スペースの代わりに改行を入れてください。

入力される値:

```text
N
```

期待する出力:

```text
A_{1,1} A_{1,2} A_{1,3} ... A_{1,N}
A_{2,1} A_{2,2} A_{2,3} ... A_{2,N}
...
A_{N,1} A_{N,2} ... A_{N,N}
```

答え:

```ruby
n = gets.to_i

(1..n).each do |i|
  row = (1..n).map do |j|
    i * j
  end

  puts row.join(' ')
end
```

解説:
`gets.to_i` で入力された自然数 `N` を整数として受け取ります。
`(1..n).each do |i|` で、1行目からN行目までを順番に作ります。
`(1..n).map do |j|` で、各行の1列目からN列目までの値を作ります。
`i * j` にすることで、i行j列目の値を計算できます。
`row.join(' ')` で数値の間を半角スペースでつなぎ、`puts` で1行ずつ出力します。

注意点:
現在のコードで `n * j` と書くと、毎回 `Nの段` だけが出力されます。
この問題では「i行j列目は i * j」なので、正しくは `i * j` です。
