# 実数を小数第M位まで丸めて出力

難易度: C
問題ID: stdout_primer__format_real_number_step4
参照元: https://paiza.jp/works/mondai/stdout_primer/stdout_primer__format_real_number_step4

問題文:
実数 N、自然数 M が入力されます。N を丸めて小数第 M 位まで出力してください。
また、N の小数部が小数第 M 位に満たない場合は 0 で埋めて出力してください。

なお、小数第 M 位が 5 になることはありません。
自然な丸め処理を行って出力すると正解になります。

入力される値:

```text
N M
```

期待する出力:

```text
N
```

末尾に改行を入れ、余計な文字、とくに `813.00000` のような余分な 0 や空行を含んではいけません。

答え:

```ruby
n, m = gets.split

m = m.to_i

puts format("%.#{m}f", n.to_f)
```

解説:
`gets.split` で、入力された `N` と `M` を分けて受け取ります。この時点では、`n` も `m` も文字列です。

`M` は小数第何位まで出力するかを表す数なので、整数に変換します。

`format("%.#{m}f", n.to_f)` で小数第 `M` 位まで出力します。
`"%.#{m}f"` は文字列展開を使って桁数を動的に指定しています。

たとえば入力が `0.813 2` なら小数第 2 位まで出力するので `0.81` になります。
入力が `0.813 4` なら小数第 4 位まで 0 で埋めるので `0.8130` になります。
