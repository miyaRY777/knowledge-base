# NがまたはNまでの数値を改行区切りで出力

難易度: D
問題ID: stdout_primer__n_line_step1

問題文:
整数 N (N = 1 または 2) が入力されます。N = 1 の場合は 1 を、N = 2 の場合は 1 と 2 を改行区切りで出力してください。

入力される値:
N

期待する出力:
答えの数値を出力してください。

N = 1 のとき

1

N = 2 のとき

1

2

末尾に改行を入れ、余計な文字、空行を含んではいけません。

答え:

```ruby
n = gets.to_i

(1..n).each do |i|
  puts i
end
```

解説:
`gets.to_i` で、入力された `N` を整数として受け取ります。
`1..n` は、1 から `n` までの範囲を表します。
`each` でその範囲の数値を1つずつ取り出し、`puts i` で改行しながら出力しています。
そのため、`N = 1` のときは `1` だけ、`N = 2` のときは `1` と `2` が改行区切りで出力されます。
