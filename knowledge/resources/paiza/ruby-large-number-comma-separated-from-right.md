# 大きな数値を右から3桁ごとカンマ区切りで出力

難易度: C
問題ID: stdout_primer__specific_split_boss

問題文:
大きな数値Nが入力されます。位の小さい方から 3 けたごとにカンマ区切りで出力してください。
ただし、Nのけた数は 3 の倍数とは限りません。

入力される値:
N

期待する出力:
答えの数値を出力してください。

M

末尾に改行を入れ、余計な文字、空行を含んではいけません。

答え:

```ruby
number = gets.chomp

puts number.reverse.scan(/.{1,3}/).join(",").reverse
```

解説:
`gets.chomp` で、大きな数値 `N` を文字列として受け取ります。
今回は「位の小さい方」、つまり右側から3けたごとに区切る必要があります。
そのため、まず `reverse` で文字列を反転させます。
`scan(/.{1,3}/)` で、反転した文字列を1〜3文字ずつ分けます。
`join(",")` でカンマ区切りにしたあと、最後にもう一度 `reverse` して元の向きに戻します。
