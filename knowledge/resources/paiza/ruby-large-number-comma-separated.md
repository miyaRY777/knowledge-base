# 大きな数値を3桁ごとカンマ区切りで出力

難易度: C
問題ID: stdout_primer__specific_split_step6

問題文:
大きな数値 N が入力されます。3 けたごとにカンマ区切りで出力してください。
ただし、N のけた数は 3 の倍数です。

入力される値:
N

期待する出力:
答えの数値を出力してください。

M

末尾に改行を入れ、余計な文字、空行を含んではいけません。

答え:

```ruby
number = gets.chomp

puts number.scan(/.{3}/).join(",")
```

解説:
`gets.chomp` で、入力された大きな数値を文字列として受け取ります。
`scan(/.{3}/)` は、文字列を3文字ずつ分けて配列にします。
例えば `123456789` なら、`["123", "456", "789"]` になります。
`join(",")` で、それらをカンマ区切りでつなげて出力します。
