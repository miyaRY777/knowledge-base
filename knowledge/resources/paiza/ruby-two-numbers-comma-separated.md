# 2つの数値をカンマ区切りで出力

難易度: D
問題ID: stdout_primer__specific_split_step1

問題文:
2 つの数値が半角スペース区切りで与えられます。これらの数値をカンマ区切りで出力してください。

入力される値:
N M

期待する出力:
答えの数値を出力してください。

N,M

末尾に改行を入れ、余計な文字、空行を含んではいけません。

答え:

```ruby
numbers = gets.chomp.split

puts numbers.join(",")
```

解説:
`gets.chomp` で、入力された1行を受け取ります。
例えば `5 1` のように、2つの数値は半角スペース区切りで入力されます。
`split` を使うと、半角スペースごとに分けて配列にできます。

```ruby
["5", "1"]
```

その後、`join(",")` を使うことで、配列の中身をカンマ区切りでつなげられます。
そのため、`5 1` が入力された場合は `5,1` と出力できます。
