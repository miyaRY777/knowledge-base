# 実数を小数第3位まで丸めて出力

難易度: D
問題ID: stdout_primer__format_real_number_step3
参照元: https://paiza.jp/works/mondai/stdout_primer/stdout_primer__format_real_number_step3

問題文:
実数 N が入力されます。N を丸めて小数第 3 位まで出力してください。
また、N の小数部が小数第 3 位に満たない場合は 0 で埋めて出力してください。

なお、小数第 4 位が 5 になることはありません。
自然な丸め処理を行って出力すると正解になります。

入力される値:

```text
N
```

期待する出力:

```text
N
```

末尾に改行を入れ、余計な文字、とくに `813.00000` のような余分な 0 や空行を含んではいけません。

答え:

```ruby
n = gets.to_f

puts format("%.3f", n)
```

解説:
`gets` で入力を受け取り、`to_f` で実数に変換します。

`format("%.3f", n)` は、数値 `n` を小数第 3 位まで表示する書き方です。

たとえば `8.13` が入力された場合、小数第 3 位まで 0 で埋めるので `8.130` と出力されます。
また `3.141692` のように小数第 4 位以降がある場合は、小数第 3 位まで丸めて `3.142` と出力されます。
