# 10個の文字列を改行区切りで出力

難易度: D
問題ID: stdout_primer__string_output_boss

問題文:
10 個の文字列 S_1, S_2, S_3, ..., S_10 が半角スペース区切りで与えられます。
これらの文字列をすべて、改行区切りで出力してください。

入力される値:
S_1 S_2 S_3 ... S_10

期待する出力:
答えの文字列を 10 行で出力してください。

S_1

S_2

S_3

...

S_10

末尾に改行を入れ、余計な文字、空行を含んではいけません。

答え:

```ruby
strings = gets.chomp.split

puts strings
```

解説:
`gets.chomp` で、入力された1行の文字列を受け取ります。
今回は10個の文字列が半角スペース区切りで1行に入力されるので、`split` を使って半角スペースごとに分けます。
`split` した結果は配列になります。
Rubyでは、配列を `puts` すると、要素を1つずつ改行して出力できます。
