# N個の実数を改行区切りで出力

難易度: D
問題ID: stdin_primer__real_number_step4

問題文:
1 行目に整数 N が与えられます。
2 行目に、N 個の実数 a_1, ... , a_N が半角スペース区切りで与えられます。
a_1, ... , a_N を改行区切りでそのまま出力してください。

入力される値:
以下の形式で標準入力によって与えられます。

N

a_1, ... , a_N

1 行目に整数 N が、2 行目に実数 a_1, ... , a_N が半角スペース区切りで与えられます。
入力値最終行の末尾に改行が 1 つ入ります。

期待する出力:
a_1, ... , a_N を改行区切りで出力してください。
また、末尾に改行を入れ、余計な文字、空行を含んではいけません。

a_1

...

a_N

答え:

```ruby
n = gets.to_i
a = gets.split.map(&:to_f)

a.each { |val| puts val }
```

解説:
1 行目の `gets.to_i` で、整数 N を受け取ります。

2 行目の `gets.split` で、半角スペース区切りの値を配列として受け取ります。

`map(&:to_f)` によって、配列の各要素を実数に変換します。

最後に `a.each { |val| puts val }` で、配列の中身を 1 つずつ取り出し、改行区切りで出力します。
