## Body
**サーバーにデータを取りに行ったり、送ったりする通信のこと**

**解説：**
`HTTPリクエスト` は、ブラウザやアプリがサーバーに対して「データをください」「このデータを保存してください」とお願いする仕組みです。
WebアプリでAPIとやり取りするときの基本になります。
`GET` や `POST` など、目的ごとに種類があります。


## Example
```js
fetch("/posts/1");
```

このコードでは、サーバーにデータ取得を依頼するために `HTTPリクエスト` を使っています。
