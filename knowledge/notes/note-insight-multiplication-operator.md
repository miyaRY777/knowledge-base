## Summary
- `*` は2つのオペランドの積を返します。
- 算数の `×` に相当しますが、プログラミングでは `*` を使います。

## Tags
#programming #javascript #operator

## Links
- [[note-insight-arithmetic-operator]]

## Body
`*` は数値の掛け算を行います。算数の `×` と区別するためアスタリスクを使います。`**` は累乗（べき乗）演算子で、乗算とは別物です。

## Example
```js
console.log(12 * 5);  // 60
console.log(2 ** 3);  // 8（累乗：2の3乗）
```
