## Summary
- 入力装置はコンピュータにデータや命令を渡すための装置です。
- キーボードやマウス、マイクなどが代表例です。
- 人がコンピュータに情報を伝える側の役割を担います。

## Tags

## Body
入力装置は、人や外部環境からの情報をコンピュータに渡す窓口です。コンピュータは自分から情報を取りに行くのではなく、まず入力装置を通じて受け取ったデータや命令をもとに動きます。何をコンピュータに伝える装置なのか、という視点で整理すると役割が理解しやすくなります。

## Example
```ruby
input_devices = ["キーボード", "マウス", "マイク"]

puts input_devices
```

このコードでは、代表的な入力装置を確認しています。
