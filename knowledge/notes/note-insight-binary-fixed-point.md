## Summary
- 2進数の固定小数点表現では、ビット列のどこまでを整数部・どこからを小数部にするかを固定します。
- 小数部の各ビットは左から `2^-1`（0.5）、`2^-2`（0.25）と重みが半分ずつになります。
- 同じビット列でも小数点位置の定義が変われば意味が変わります。

## Body
2進数で小数を表すには、整数部と小数部にビットを割り当てます。たとえば `1010.1100` なら整数部 `1010`、小数部 `.1100` です。小数部のビットは左から `2^-1 = 0.5`、`2^-2 = 0.25`、`2^-3 = 0.125`、`2^-4 = 0.0625` の重みを持ちます。「上位4ビットを整数部、下位4ビットを小数部」のように仕様で決めた位置に小数点を置いて読むことが固定小数点表現の本質です。

## Example
```
1010.1100₂

整数部：1×2^3 + 0×2^2 + 1×2^1 + 0×2^0 = 10
小数部：1×2^-1 + 1×2^-2 = 0.5 + 0.25 = 0.75

→ 10進数で 10.75
```
