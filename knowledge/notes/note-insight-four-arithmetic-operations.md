## Summary
- 四則演算は数値計算の基本となる加算・減算・乗算・除算の4種類です。
- プログラミングでは `+`・`-`・`*`・`/` で表します。

## Body
算数の四則演算と基本的に同じですが、掛け算は `×` でなく `*`、割り算は `÷` でなく `/` を使います。JavaScriptでは整数も小数も `number` 型なので、`10 / 3` は `3.3333...` になります。

## Example
```js
console.log(10 + 2); // 12
console.log(10 - 2); // 8
console.log(10 * 2); // 20
console.log(10 / 2); // 5
```
