## Summary
- 入力・処理・出力は、プログラムの動きを理解するための基本的な考え方です。
- 入力データを受け取り、処理して、必要に応じて出力データを作る3段階で整理できます。
- 特定技術に限らず、プログラム全般に共通する概念です。

## Body
「入力・処理・出力」はプログラムの動きを整理する最もシンプルな枠組みです。どんな複雑なプログラムも、この3段階に分解して考えることができます。何を入力として受け取り、何を処理し、何を出力として返すか——この3点を意識するだけで、関数やメソッドの設計が明確になります。

## Example
```java
// 入力: a, b　処理: 足し算　出力: 合計値
public static int add(int a, int b) {
    return a + b;
}
```
