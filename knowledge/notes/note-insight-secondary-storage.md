## Summary
- 補助記憶装置はデータを長く保存しておくための装置です。
- HDD や SSD が代表例です。
- 主記憶装置より遅いですが、電源を切っても内容が残ります。

## Tags
#computer #hardware #storage

## Links
- [[note-insight-storage-device]]

## Body
補助記憶装置は、ファイルやアプリケーション、長期的に残したいデータを保存するための場所です。主記憶装置ほど高速ではありませんが、電源を切っても内容が消えないため、継続して保持したい情報の保存先として使われます。長く置いておく倉庫のような役割と考えると理解しやすいです。

## Example
```ruby
secondary_storage = ["SSD", "HDD"]

puts secondary_storage
```

このコードでは、代表的な補助記憶装置を確認しています。

## Action
- [ ] SSD と HDD の違いも別ノート候補として切り出す
