## Summary
- 記憶装置はデータやプログラム、処理結果を保存するための装置です。
- 主記憶装置と補助記憶装置に大きく分けられます。
- 一時的に使う場所か長く保存する場所かで役割が分かれます。

## Body
記憶装置は、コンピュータが扱うデータやプログラムを保持する役割を持ちます。処理の途中で一時的に使う高速な場所と、長く保存しておく場所では性質が違うため、主記憶装置と補助記憶装置に分けて理解すると整理しやすくなります。保存の速さや電源を切ったあとの状態も重要な違いです。

## Example
```ruby
storage_types = {
  main_memory: "メモリ",
  secondary_storage: "ストレージ"
}

puts storage_types
```

このコードでは、記憶装置の種類を整理しています。
