## Summary
- 各桁の `1` に対応する `2の累乗` を足すと10進数に変換できます。
- 整数部は右端から `2^0`、`2^1`、`2^2` と増え、小数部は左から `2^-1`、`2^-2` と減ります。
- 小数部を含む場合は整数部と小数部を別々に計算して合計します。

## Body
2進数の各桁は右から `2^0=1`、`2^1=2`、`2^2=4`、`2^3=8` の重みを持ちます。その桁が `1` なら重みを足し、`0` なら足しません。小数部は左から `2^-1=0.5`、`2^-2=0.25` と半分ずつ小さくなります。この仕組みを理解すると固定小数点や浮動小数点の読み方にも応用できます。

## Example
```
1011.01₂ を10進数に変換する

整数部 1011：
  1×2^3 = 8
  0×2^2 = 0
  1×2^1 = 2
  1×2^0 = 1

小数部 .01：
  0×2^-1 = 0
  1×2^-2 = 0.25

合計：8 + 0 + 2 + 1 + 0 + 0.25 = 11.25
```
