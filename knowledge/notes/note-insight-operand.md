## Summary
- オペランドは、演算子の処理対象となる値です。
- `8 + 3` では `8` と `3` がオペランドです。

## Tags
#programming #operator

## Links
- [[note-insight-operator]]
- [[note-insight-expression]]

## Body
演算子とオペランドを組み合わせることで式を作ります。2つのオペランドをとる演算子を二項演算子と呼びます。

## Example
```js
console.log(8 + 3); // 8と3がオペランド、+が演算子
```
