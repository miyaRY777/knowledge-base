* [ ] `オペランド`とは？

👉 **演算子によって処理される値**

**解説：**
`オペランド`は、足し算や引き算などの処理の対象になる値です。
`8 + 3`では、`8`と`3`がオペランドです。

```js
console.log(8 + 3);
```

このコードでは、加算の対象になる`8`と`3`が`オペランド`です。

---

* [ ] `演算子`とは？

👉 **値に対して特定の処理を行う記号**

**解説：**
`演算子`は、オペランドに計算や比較などの処理を適用し、新しい値を返します。
例えば、`+`は加算、`-`は減算を行う演算子です。

```js
console.log(8 + 3);
```

このコードでは、`8`と`3`を足すために加算`演算子`の`+`を使っています。

---

* [ ] `式`とは？

👉 **評価すると1つの値になるコードのまとまり**

**解説：**
`式`は、値や演算子などを組み合わせたもので、実行すると1つの値になります。
例えば、`8 + 3`は評価されると`11`になります。

```js
console.log(8 + 3);
```

このコードでは、`8 + 3`という`式`が`11`として評価されます。

---

* [ ] `評価`とは？

👉 **式を計算して結果の値を決めること**

**解説：**
`評価`とは、プログラムが式の内容を処理し、その結果となる値を求めることです。
式そのものではなく、評価された結果が次の処理で使われます。

```js
const result = 8 + 3;

console.log(result);
```

このコードでは、`8 + 3`を`評価`した結果の`11`を`result`に代入しています。

---

* [ ] `算術演算子`とは？

👉 **数値の計算を行う演算子**

**解説：**
`算術演算子`は、足し算、引き算、掛け算、割り算などを行うために使います。
代表的なものには、`+`、`-`、`*`、`/`があります。

```js
console.log(12 + 5);
console.log(12 - 5);
console.log(12 * 5);
console.log(12 / 5);
```

このコードでは、数値の四則演算を行うために`算術演算子`を使っています。

---

* [ ] `四則演算`とは？

👉 **足し算・引き算・掛け算・割り算の4つの計算**

**解説：**
`四則演算`は、数値計算の基本となる4種類の演算です。
プログラミングでは、`+`、`-`、`*`、`/`を使って表します。

```js
console.log(10 + 2);
console.log(10 - 2);
console.log(10 * 2);
console.log(10 / 2);
```

このコードでは、4種類の基本的な計算を行うために`四則演算`を使っています。

---

* [ ] `加算演算子（+）`とは？

👉 **2つの値を足す演算子**

**解説：**
`加算演算子`は、数値同士を足して新しい値を返します。
JavaScriptでは文字列に使用すると、文字列をつなげる処理になるため注意が必要です。

```js
console.log(12 + 5);
// 17
```

このコードでは、`12`と`5`を足すために`加算演算子（+）`を使っています。

---

* [ ] `減算演算子（-）`とは？

👉 **左側の値から右側の値を引く演算子**

**解説：**
`減算演算子`は、2つの数値の差を求めるために使います。
計算する順番によって結果の正負が変わります。

```js
console.log(12 - 5);
// 7
```

このコードでは、`12`から`5`を引くために`減算演算子（-）`を使っています。

---

* [ ] `乗算演算子（*）`とは？

👉 **2つの値を掛ける演算子**

**解説：**
`乗算演算子`は、数値同士の掛け算を行います。
プログラミングでは、算数で使う`×`ではなく`*`を使います。

```js
console.log(12 * 5);
// 60
```

このコードでは、`12`と`5`を掛けるために`乗算演算子（*）`を使っています。

---

* [ ] `除算演算子（/）`とは？

👉 **左側の値を右側の値で割る演算子**

**解説：**
`除算演算子`は、数値の割り算を行います。
整数同士を割ったときの結果は、プログラミング言語やデータ型によって異なります。

```js
console.log(12 / 5);
// 2.4
```

このコードでは、`12`を`5`で割るために`除算演算子（/）`を使っています。

---

* [ ] `整数同士の割り算`とは？

👉 **整数型の値だけを使って行う割り算**

**解説：**
整数同士の割り算で小数部分を残すかどうかは、言語によって異なります。
JavaScriptでは`12 / 5`は`2.4`ですが、JavaやC++で整数型同士を割ると、一般的に小数部分が切り捨てられて`2`になります。

**具体例：**

* JavaScriptの`12 / 5`は`2.4`
* Javaの`12 / 5`は、両方が`int`なら`2`
* Javaの`12.0 / 5.0`は`2.4`
* 言語とオペランドのデータ型を確認する必要がある

この例では、割り算の結果を正しく理解するために`整数同士の割り算`の仕様が関係しています。

---

* [ ] `整数除算`とは？

👉 **割り算の結果から小数部分を除き、整数として扱う計算**

**解説：**
`整数除算`では、割り切れない部分を整数の範囲で処理します。
どの方向へ丸めるかや使用する演算子は、言語によって異なるため注意が必要です。

```java
int result = 12 / 5;

System.out.println(result);
// 2
```

このコードでは、`int`型同士で割り算を行うため、`整数除算`の結果として`2`が返されます。

---

* [ ] `演算結果のデータ型`とは？

👉 **式を評価した結果が属するデータ型**

**解説：**
演算結果には値だけでなく、整数型や浮動小数点型などのデータ型もあります。
結果の型は、使用する言語、演算子、オペランドの型によって決まります。

```js
const result = 12 / 3;

console.log(result);
console.log(typeof result);
```

このコードでは、割り算の結果と、その`演算結果のデータ型`を確認しています。

* [ ] `Infinity`とは？

👉 **JavaScriptで無限大を表す特別な数値**

**解説：**
JavaScriptでは、正の数を`0`で割ると`Infinity`が返されます。
`Infinity`はエラー文字列ではなく、`number`型に属する特別な値です。

```js
console.log(1 / 0);
console.log(typeof Infinity);
```

このコードでは、正の数を`0`で割った結果として`Infinity`が返されます。

---

* [ ] `-Infinity`とは？

👉 **JavaScriptで負の無限大を表す特別な数値**

**解説：**
JavaScriptでは、負の数を正の`0`で割るなどの計算によって`-Infinity`が返される場合があります。
`Infinity`と同じく、`number`型の特別な値です。

```js
console.log(-1 / 0);
// -Infinity
```

このコードでは、負の数を`0`で割った結果として`-Infinity`が返されます。

---

* [ ] `NaN`とは？

👉 **数値として有効な結果を表せないことを示す特別な値**

**解説：**
`NaN`は`Not a Number`の略で、数値計算の結果を有効な数として表せない場合に使われます。
JavaScriptでは`0 / 0`などの計算によって返されます。

```js
console.log(0 / 0);
console.log(Number.isNaN(0 / 0));
```

このコードでは、数値として結果を決められない`0 / 0`によって`NaN`が返されます。

---

* [ ] `0除算`とは？

👉 **数値を0で割る計算**

**解説：**
`0除算`を行ったときの動作は、プログラミング言語やデータ型によって異なります。
JavaScriptでは`1 / 0`が`Infinity`になりますが、言語によってはエラーが発生します。

**具体例：**

* JavaScriptの`1 / 0`は`Infinity`
* JavaScriptの`0 / 0`は`NaN`
* Javaの整数型で`1 / 0`を行うと例外が発生する
* 言語によって結果が違うため、事前に`0`でないか確認する

この例では、意図しない結果やエラーを防ぐために`0除算`への注意が必要です。

---

* [ ] `複数の演算子を含む式`とは？

👉 **1つの式の中で複数の計算を行う式**

**解説：**
複数の演算子がある式は、決められた優先順位や結合法則に従って評価されます。
同じ優先順位の加算と減算は、通常は左から右へ処理されます。

```js
console.log(5 + 10 - 5);
// 10
```

このコードでは、最初に`5 + 10`が評価され、その結果から`5`を引いています。

---

* [ ] `オペランドのデータ型`とは？

👉 **演算の対象となる値が属しているデータ型**

**解説：**
同じ演算子でも、オペランドのデータ型によって処理や結果が変わる場合があります。
演算を行うときは、値だけでなく整数型、浮動小数点型、文字列型などの型も確認する必要があります。

```js
console.log(2 + 5);
console.log("2" + 5);
```

このコードでは、オペランドが数値なら加算され、文字列を含む場合は文字列の連結になります。
