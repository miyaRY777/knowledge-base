# 2026-06-23 Ruby 数値の丸めメソッド

## round メソッド

👉 **数値を指定した桁数で四捨五入するRubyのメソッド**

引数に数字を渡すと、小数点以下の桁数を指定できる。
Ruby公式: `Float#round` は指定した小数桁数の精度で近い値に丸めるメソッド。

```ruby
num = 100.3383
puts num.round(2)
# => 100.34
```

---

## ceil メソッド

👉 **数値を指定した桁数で切り上げるRubyのメソッド**

引数なしでは整数に切り上げ、引数を指定すると小数点以下の桁数を指定して切り上げできる。
Ruby公式: `Float#ceil` は自分自身以上の最小の数を返すメソッド。

```ruby
num = 100.3333
puts num.ceil(1)
# => 100.4
```

---

## floor メソッド

👉 **数値を指定した桁数で切り捨てるRubyのメソッド**

引数なしでは整数に切り捨て、引数を指定すると小数点以下の桁数を指定して切り捨てできる。
Ruby公式: `Float#floor` は自分自身以下の最大の数を返すメソッド。

```ruby
num = 100.3383
puts num.floor(3)
# => 100.338
```
