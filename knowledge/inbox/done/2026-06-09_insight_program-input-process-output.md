* [ ] `プログラム`とは？



👉 **コンピュータにやってほしい処理を書いた命令の集まり**



**解説：**

`プログラム` は、コンピュータに「何をしてほしいか」を伝えるための手順です。

計算する、文字を表示する、データを保存するなど、目的に合わせて命令を書きます。

多くのプログラムは、入力を受け取り、処理して、必要に応じて出力を作ります。



**具体例：**



* 数字を受け取って計算する

* 画面に文字を表示する

* フォームの入力内容を保存する

* 画像や一覧を画面に表示する



この例では、コンピュータに特定の作業をさせるために `プログラム` が関係しています。



---



* [ ] `入力データ`とは？



👉 **プログラムが処理するために受け取るデータ**



**解説：**

`入力データ` は、プログラムが処理を始めるために受け取る材料のようなものです。

ユーザーが入力した文字、数値、ファイル、フォームの内容などが入力データになります。

入力データがあることで、プログラムはそれをもとに計算や判定を行えます。



**具体例：**



* 足し算プログラムに `3` と `5` を渡す

* ログイン画面でメールアドレスとパスワードを入力する

* 検索フォームにキーワードを入力する



この例では、プログラムが処理を始める材料として `入力データ` が関係しています。



---



* [ ] `処理`とは？



👉 **入力データを使って、計算・判定・保存などを行うこと**



**解説：**

`処理` は、プログラムが実際に行う作業のことです。

入力されたデータを計算したり、条件に合うか判定したり、データベースに保存したりします。

特定技術だけの特別な機能名ではなく、プログラム全般で使われる一般的な考え方です。



**具体例：**



* `3 + 5` を計算する

* パスワードが正しいか確認する

* 入力されたユーザー情報をデータベースに保存する

* 商品一覧を並び替える



この例では、入力されたデータに対して作業を行うために `処理` が関係しています。



---



* [ ] `出力データ`とは？



👉 **プログラムが処理した結果として返すデータ**



**解説：**

`出力データ` は、プログラムが処理を終えたあとに作る結果です。

計算結果、検索結果、画面に表示する文字、APIのレスポンスなどが出力データになります。

すべてのプログラムが出力データを返すわけではありません。



**具体例：**



* `3 + 5` の結果として `8` を返す

* 検索結果の一覧を表示する

* APIがJSON形式でユーザー情報を返す



この例では、処理結果を外に渡すために `出力データ` が関係しています。



---



* [ ] `入力・処理・出力`とは？



👉 **プログラムの基本的な流れ**



**解説：**

`入力・処理・出力` は、プログラムの動きを理解するための基本的な考え方です。

入力データを受け取り、それを処理して、必要に応じて出力データを作ります。

特定技術だけの特別な機能名ではなく、プログラム全般を理解するための考え方です。



```java

public static int add(int a, int b) {

    return a + b;

}

```



このコードでは、`a` と `b` を入力として受け取り、足し算を処理し、結果を返すために `入力・処理・出力` が関係しています。



---



* [ ] `出力データを作成するプログラム`とは？



👉 **処理した結果を値として返すプログラム**



**解説：**

`出力データを作成するプログラム` は、入力データを処理して、結果を呼び出し元へ返します。

Javaでは、返す値がある場合、`int` や `String` など戻り値の型を指定します。

OracleのJavaチュートリアルでも、値を返すメソッドは戻り値の型を宣言すると説明されています。 ([ウィキペディア][1])



```java

public static int add(int a, int b) {

    return a + b;

}

```



このコードでは、足し算の結果を `int` の値として返すために `出力データを作成するプログラム` が関係しています。



---



* [ ] `出力データを作成しないプログラム`とは？



👉 **結果の値を返さず、処理だけを行うプログラム**



**解説：**

`出力データを作成しないプログラム` は、画面表示や保存処理など、処理を実行すること自体が目的です。

Javaでは、値を返さないメソッドに `void` を指定します。

`void` はメソッドが値を返さないことを宣言するキーワードです。 ([ウィキペディア][2])



```java

public static void sayHello() {

    System.out.println("Hello World");

}

```



このコードでは、文字を表示するだけで値を返さないために `出力データを作成しないプログラム` が関係しています。



---



* [ ] `戻り値`とは？



👉 **関数やメソッドの処理結果として返される値**



**解説：**

`戻り値` は、関数やメソッドを実行したあと、呼び出し元に返す結果です。

計算結果や取得したデータを次の処理で使いたいときに戻り値を返します。

Javaでは、戻り値がある場合は戻り値の型を指定し、戻り値がない場合は `void` を指定します。 ([ウィキペディア][1])



```java

public static int getAge() {

    return 20;

}

```



このコードでは、`20` という数値を呼び出し元に返すために `戻り値` を使っています。



---



* [ ] `void`とは？



👉 **関数やメソッドが値を返さないことを示すキーワード**



**解説：**

`void` は、Javaなどで「このメソッドは戻り値を返さない」と示すために使います。

`int` や `String` のように値の型を表すのではなく、返す値がないことを表します。

Javaでは、値を返さないメソッドの戻り値の型として `void` を書きます。 ([ウィキペディア][2])



```java

public static void greet() {

    System.out.println("こんにちは");

}

```



このコードでは、画面に文字を表示するだけで値を返さないために `void` を使っています。
