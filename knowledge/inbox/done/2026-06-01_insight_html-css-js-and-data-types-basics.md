`CSS → プロパティ: 値` は、どの見た目を、どのように変えるかを指定する書き方です。
`color: red;` なら、`color` がプロパティ、`red` が値です。
MDNでは、CSS宣言はプロパティと値の組み合わせとして説明されています。([MDNウェブドキュメント](https://developer.mozilla.org/en-US/docs/Web/API/CSS_Object_Model/CSS_Declaration?utm_source=chatgpt.com "CSS Declaration - Web APIs | MDN"))

```css
p {
  color: red;
}
```

このコードでは、文字色を赤にするために `CSS → プロパティ: 値` の形を使っています。

- [ ] `HTMLの属性`とは？

👉 **HTML要素に追加の設定や情報を与えるもの**

**解説：**
`HTMLの属性` は、HTML要素に対して、動作・見た目の目印・リンク先・入力欄の種類などを指定するために使います。
MDNでは、HTML要素には属性があり、要素を構成したり動作を調整したりする追加の値だと説明されています。([MDNウェブドキュメント](https://developer.mozilla.org/ja/docs/Web/HTML/Reference/Attributes?utm_source=chatgpt.com "HTML 属性リファレンス - MDN Web Docs"))
属性によって使える要素や意味が異なるため、適切な属性を使うことが大切です。

```html
<input type="text" name="username">
```

このコードでは、入力欄の種類と送信時の名前を指定するために `HTMLの属性` を使っています。
